Explorando el Diseño Web

¿Qué es el Diseño Web?

El diseño web es el proceso de crear y organizar páginas en internet. Combina creatividad y tecnología para construir sitios atractivos y funcionales.

Herramientas que estoy aprendiendo

HTML: Estructura el contenido de una página web.

CSS: Da estilo y diseño visual.
